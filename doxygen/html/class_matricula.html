Padrão para representação da classe Matricula.

Regras de formato:

  • Matricula válida somente se a quantidade de dígitos não for maior que 5.
  • Matricula válida somente se não houver dígitos duplicados.
  • Matricula válida somente se estiver no formato XXXXX onde cada X é um dígito (0-9).
